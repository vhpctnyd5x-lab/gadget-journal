# Gadget Journal

Gadget Journalは、ガジェットに関する記録・共有のためのジャーナルアプリケーションです。

## 主な機能
- ガジェットの登録・編集・削除
- ガジェット情報の一覧表示
- 写真や説明の追加

## インストール方法
1. このリポジトリをクローン
2. 必要に応じて`npm install`や`yarn install`を実行

## 使用方法
- `index.html`をブラウザで開く、またはローカルサーバー上で実行
