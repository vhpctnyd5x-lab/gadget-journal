# MacBook Neo — Appleが「再発明」するノートブック

2006年にMacBook Pro初代が登場し、ノートブックをアルミニウムで再構築。2008年にMacBook Airが「世界最薄ノートPC」を実現し、一般的なノートブック概念を覆した。2020年にApple Siliconで「なぜIntelを使っていたのか」と思わせ、バッテリー効率を2倍に。

Appleはノートブック市場を5年ごとに再定義してきた。では**MacBook Neo**——複数の信頼性の高いアナリストが2027〜2028年の登場を予測する「次世代MacBook」のコードネーム——が噂される中、次の再発明はどこへ向かうのか。本記事では、リーク情報と特許、サプライチェーン分析を元に、MacBook Neoの正体に迫る。

## MacBook Neoとは何か

### 正式発表はまだ

「MacBook Neo」はAppleが正式発表した製品名ではない。リーク情報、特許取得、サプライチェーン情報を元に、業界アナリストが名付けた次世代MacBookの開発コード的な呼称。

信頼性の高い複数の情報源が言及：

| 情報源 | 信頼度 | 予測時期 |
|--------|--------|---------|
| Ming-Chi Kuo（アナリスト） | ⭐⭐⭐⭐⭐ | 2027-2028年登場 |
| Mark Gurman（Bloomberg記者） | ⭐⭐⭐⭐⭐ | 2027年プロトタイプ |
| Apple Supply Chain報告 | ⭐⭐⭐⭐ | 2027年試作着手 |
| 特許情報（USPTO） | ⭐⭐⭐ | フォルダブルディスプレイ技術確認 |

### 現在のMacBook製品ラインアップ

| モデル | 画面 | チップ | 主な用途 |
|--------|------|--------|--------|
| MacBook Air M5 | 13.6/15.3"液晶 | M5 | 一般向け・学生 |
| MacBook Pro 14/16" | 14.2/16"ミニLED | M5/M5 Pro/Pro Max | プロフェッショナル |
| **MacBook Neo（予測）** | フォルダブルOLED or ガラスキーボード | M6/M7 | 次世代ユーザー |

MacBook Neoは「新たなカテゴリー」として提案されるだろう——MacBook Airより低価格で、Pro以上の革新性を持つ「未来のノートブック」として。

## Tandem OLED：光学技術の革命

### Liquid Retina XDR vs OLED

現在のMacBook Proはミニ LED（バックライト）を使った**Liquid Retina XDRディスプレイ**。美しいが、OLEDにはまだ到達していない。

| 項目 | Liquid Retina XDR | Tandem OLED |
|------|-------------------|-------------|
| 技術 | ミニLED（1,152個）+ 液晶層 | 有機EL 2枚重ね |
| 黒の表現 | 非常に深い（完全黒ではない） | 完全な黒（ピクセルオフ） |
| コントラスト比 | 1,000,000:1 | ∞（理論値） |
| 応答時間 | 数ms | <1ms |
| バックライト | あり | なし |
| バッテリー効率 | 基準 | Dark Mode時30%改善 |
| 色精度 | DCI-P3 100% | DCI-P3 100% + 広色域 |
| 焼き付き | なし | あり（対策技術進化） |
| 開発段階 | 成熟期 | 成長期（iPad Pro M5で初採用） |

### Tandem OLED技術詳細

iPad Pro 13インチが2024年に世界初搭載した**Tandem OLED**は、2枚のOLEDを重ねた構造：

**構成**：
1. 上層OLED（マスターディスプレイ）
2. 下層OLED（バックライト代替）
3. 透光性の中間層

**メリット**：
- **輝度2倍**：各層で輝度を発揮し、組み合わせで1,600ニト超実現（従来OLEDは600ニト程度）
- **焼き付き耐性向上**：各ピクセルへの駆動負荷が分散
- **バッテリー効率**：暗い表示では下層のみ動作させることで大幅節電

### MacBook Neoへの適用

MacBook Neoで**Tandem OLED**が採用されるとすれば、MacBook Proを超える画質が実現される：

| 仕様 | 予測値 |
|------|--------|
| ディスプレイサイズ | 14.2"/16"予測 |
| 解像度 | 3456×2234 / 3840×2496（スケール） |
| リフレッシュレート | 120Hz ProMotion |
| 最大輝度 | 1,600ニト（標準）/ 2,000ニト（HDR） |
| 色精度 | DCI-P3 100%以上 |
| バッテリー向上 | +15〜25% |

### フォルダブルMacBookのシナリオ

#### シナリオ1：縦折りタイプ（上下2段）

| 項目 | 仕様 |
|------|------|
| 折り状態 | iPad Pro 13"サイズ（折りたたみ）|
| 展開状態 | ノートPC（キーボード+ディスプレイ） |
| キーボード | オンスクリーンキーボード or 外付け |
| 厚さ | 折り時5mm程度、展開時8mm |
| 用途 | 携帯性と大画面のバランス |

**メリット**：
- iPad並みのコンパクト性で持ち運べる
- 展開時は16"相当の大画面
- 外出時はタブレット、オフィスでノートPC

**デメリット**：
- 折り線（シーム）の視認性
- 物理キーボードの廃止
- 価格の上昇（フォルダブル技術は高価）

#### シナリオ2：縦折りタイプ（下部キーボード固定）

| 項目 | 仕様 |
|------|------|
| 下部 | 物理キーボード（固定） |
| 上部 | フォルダブルOLED（折り曲げ可） |
| 用途 | 画面サイズを状況に応じて変更 |
| ユースケース | ビジネス資料とキーボードの両立 |

**利点**：
- 物理キーボード維持（タイピング感保持）
- 画面をコンパクト→大画面に展開できる柔軟性
- 1台で複数の作業スタイルに対応

## ガラスキーボード：タクタイル革命

### 仮想キーボードの最終形

最も大胆な噂の一つが**ガラスキーボード**。物理的なキーメカニズムを廃止し、フラットなガラス面にキーが**ホログラフィック表示**または**LEDマトリックス表示**される。

### 技術的実装

#### Taptic Engine進化版

Appleは既にMagic Trackpadで「実際にはクリックしていない」のに「クリック感覚」を生み出す触覚フィードバック技術を完成。

MacBook Neoのガラスキーボードなら、キーボード全面に**ナノ精度の振動触覚フィードバック**を搭載：

| 機能 | 説明 |
|------|------|
| キータッチ感覚 | ボタン押下時に各キー下で微細振動 |
| 押下感の深さ調整 | キーボード設定で押下ストローク調整可 |
| フィードバック強度 | ゲーミング/執筆など用途別に最適化 |
| アダプティブ | 手の位置・気分・時間帯に基づいて自動調整 |

### ガラスキーボードのメリット

#### 1. レイアウト動的変更

| 用途 | キー配置 |
|------|--------|
| 日本語入力 | あいうえお配列 |
| 英語入力 | QWERTY配列 |
| プログラミング | 括弧・セミコロン優先 |
| デザイン | ショートカット専用キー |

ユーザーが入力言語やアプリケーションを切り替えると、キーボード配置が瞬時に変更される——これは物理キーボードには不可能。

#### 2. マクロキーの自由度

| シーン | 活用例 |
|--------|--------|
| Photoshop作業 | レイヤー操作キーをカスタム配置 |
| コーディング | よく使う関数名をマクロキー化 |
| 日本語入力 | よく使う変換語を単一キー化 |

#### 3. 薄型化

物理キーメカニズムがなくなり、本体厚さを現行比30〜40%削減可能。

### ガラスキーボードの課題

#### 1. タイピング習慣の変更抵抗

ユーザーは物理キーの「クリック感」に慣れている。ガラスキーボードへの移行には心理的抵抗あり。

Appleの解決策：
- **段階的フィードバック**：初期段階では強い振動→ユーザー慣れたら調整
- **教育キャンペーン**：メディア、YouTuber向けハンズオン企画
- **オプション外付けキーボード**：移行期は物理キーボード併用可

#### 2. 長期耐久性の未知

ガラスフラット表面への摩耗、Taptic Engineの寿命期待値、修理可能性——すべて未知数。

Appleの歴史から言えば、十分な品質保証テスト（通常5年×100万回使用シミュレーション）を経た後の発表になるはず。

#### 3. 価格上昇

Tandem OLED + カスタマイズガラスキーボード + Taptic Engineフル搭載 → 製造コスト上昇 → 定価も上昇

#### 4. キー誤操作対策

ガラスフラット表面で、誤タッチが増える可能性。解決案：

| 対策 | 説明 |
|------|------|
| パームリジェクション | 手のひら領域をAIで検出して入力無視 |
| キーボードロック感知 | 手がキーボード上を移動中は誤操作判定 |
| ジェスチャー入力 | ダブルタップでオン/オフ |

## 業界への影響

### PC市場への波紋

MacBook Neo発表は、Windows PCメーカーに衝撃：

- **Dell XPS**：フォルダブルディスプレイ競争
- **Microsoft Surface**：物理キーボード廃止検討
- **HP/Lenovo**：コピー製品の同時進行

### Appleの戦略的意図

「なぜ今、フォルダブルMacBook？」という問いへの答え：

1. **新しい用途提案**：iPad と MacBook の「中間デバイス」化
2. **プライバシー強化**：ガラスキーボードでロギング不可能
3. **クリエイター層の再掘り込み**：高価格帯での利益率向上
4. **技術的優位の維持**：Apple Silicon + Tandem OLED + フォルダブルの唯一無二の組み合わせ

## まとめ

MacBook Neoはまだコンセプトの域を出ないが、Appleがノートブックの「次の形」を真摯に模索していることは明白。Tandem OLEDディスプレイ、ガラスキーボード、フォルダブル化——これらのうちどれが実現するかはわからないが、「MacBookが大きく変わる日」は確実に近づいている。

一つ確かなのは、Appleは「薄さのため」だけの変化をしないこと。ガラスキーボードなら多言語対応とマクロキー自由度、フォルダブルなら携帯性と大画面の両立——何らかの「理由ある変革」として登場するはずだ。

2027年のMacBook Neoの正式発表は、ノートブック業界全体の転換点になるだろう。
