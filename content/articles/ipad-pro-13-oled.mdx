# iPad Pro 13インチ OLED — デジタルアートの最高峰

iPad Pro 13インチOLEDは、デジタルクリエイターの夢を実現した。13インチの広大なTandem OLEDキャンバス、M5チップのパワー、Apple Pencil Proとの完全統合。これはタブレットではなく、ポータブルスタジオだ。

## Tandem OLED：ディスプレイ革命

iPad Pro M5のTandem OLEDは、2枚のOLEDパネルを重ねて駆動する独自技術：

| 仕様 | iPad Pro 13インチ M5 | iPad Air M5 |
|------|--------------------|-----------| 
| ディスプレイ技術 | Tandem OLED | Liquid Retina |
| 解像度 | 2752×2064（264ppi） | 2732×2048（264ppi） |
| コントラスト比 | 無限大（完全黒が可能） | 500:1 |
| SDR輝度（持続） | 1,000ニト | 600ニト |
| HDR輝度（ピーク） | 1,600ニト | なし |
| リフレッシュレート | 最大120Hz（ProMotion） | 最大120Hz（ProMotion） |
| 局所調光 | ピクセルレベル | パネル全体 |

Tandem OLEDの「ピクセルレベル調光」は実際の制作に大きな差を生む。暗い背景に明るい被写体がある写真のレタッチで、OLEDの完全な黒と鮮やかな白が同一フレームに共存できる。

## M5チップ：iPadに載ったMacBook Pro相当の処理能力

iPad Pro M5搭載のM5チップは、16GBのユニファイドメモリを搭載：

| 仕様 | iPad Pro M5 | iPad Air M5 |
|------|------------|------------|
| チップ | M5 | M5 |
| CPU | 10コア | 10コア |
| GPU | 10コア | 10コア |
| Neural Engine | 38TOPS | 38TOPS |
| RAM | 16GB | 8GB |
| 最大ストレージ | 2TB | 1TB |

RAMが16GBあることで、複数のプロアプリを同時に起動したままにできる。ProcreateとAdobe Frescoを両方開いた状態でスイッチしても、データが失われない。

## Apple Pencil Pro：ペンの究極形

iPad Pro 13インチとApple Pencil Proの組み合わせは、デジタルアートの最高峰だ：

| 機能 | 動作 |
|------|------|
| バレルロール検出 | ペンを回転すると筆の向きが変化 |
| スクイーズジェスチャー | 軽く握るとツール切り替え |
| ホバー検出 | 12mm手前でカーソルが追従 |
| 触覚フィードバック | ペン先からリアルな振動感 |
| ProMotion 120Hz追従 | ペンの動きが画面に即時反映 |
| 傾き検知 | 傾けてブラシの太さを変更 |

Procreateでバレルロールを使った筆さばきは、水彩や油絵の自然な動きを再現する。プロのイラストレーターが「物理的なカンバスは要らない」と言う理由だ。

## iPad Air M5との差：どちらを選ぶか

| 比較 | iPad Pro 13インチ M5 | iPad Air 13インチ M5 |
|------|-------------------|--------------------|
| 価格（Wi-Fi/128GB） | ¥178,800 | ¥138,800 |
| ディスプレイ | Tandem OLED | Liquid Retina |
| RAM | 16GB | 8GB |
| 最大ストレージ | 2TB | 1TB |
| Face ID | 横向き対応 | なし（Touch ID） |
| LiDARスキャナー | あり | なし |
| 厚さ | 5.1mm | 6.1mm |
| 価格差 | +¥40,000 | ベース |

**iPad Pro 13インチを選ぶ理由**：
1. OLEDの黒の深さ・HDR輝度が必要（写真現像・映像制作）
2. 16GBのRAMで複数重いアプリを同時起動したい
3. LiDARスキャナーを使うARアプリ・3Dスキャンを使う
4. 2TBストレージが必要

**iPad Air 13インチで十分な理由**：
1. イラスト・ドキュメント・学習が主な用途
2. 予算を抑えたい（4万円の節約）
3. 120HzのProMotionは両モデル搭載済みで体験は近い
