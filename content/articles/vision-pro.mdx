# Apple Vision Pro — これはコンピューターの再発明だ

「初めてMacを使った時の感覚」と表現する人がいる。「初めてiPhoneを使った時の衝撃」と語る人もいる。Apple Vision Proをかけた瞬間、その言葉の意味を体で理解した。これは**コンピューターとの関係性を根本から変える装置**だ。

¥499,800という価格、600gの重量、2時間のバッテリー——制約は確かにある。しかし片目4K相当（合計23M画素）のマイクロOLEDが目の前に広がる映像、視線で選択し指でタップする操作、空間に浮かぶウィンドウ——これらは「未来のコンピューター」を現実に体験させてくれる、現時点で世界唯一のデバイスだ。

## Apple Vision Proとは

### 製品定義

Apple Vision Proは、Appleが開発した**空間コンピューター**：

| 分類 | Vision Pro | 一般的VRゴーグル |
|------|-----------|---------------|
| 用途 | 空間コンピューティング | ゲーム・VRコンテンツ |
| 現実視認 | パススルーで現実が見える | 現実遮断 |
| 操作 | 視線+ジェスチャー+音声 | コントローラー |
| OS | visionOS（Apple独自） | Android系 |
| 用途範囲 | PC、エンタメ、コミュニケーション | 主にエンタメ |
| 価格帯 | ¥499,800〜 | ¥30,000〜¥150,000 |

### Mixed Reality（複合現実）の本質

VRが現実世界を完全に遮断するのに対し、Vision Proは現実世界の上にデジタルコンテンツを重ねる**複合現実**装置。

部屋の中にいながら：
- 空間に複数のウィンドウを浮かべて作業
- 壁に巨大スクリーンを配置して映画鑑賞
- テーブルに3Dモデルを置いてデザイン
- 同じ空間にFaceTime相手のホログラム

## ディスプレイ：23M画素のマイクロOLED

### 技術仕様

| 項目 | 仕様 |

### 比較データ

| デバイス | 総画素数 |
|--------|--------|
| Apple Vision Pro | 23M画素 |
| MacBook Pro 16" | 約10M画素 |
| iPhone 17 Pro Max | 約4M画素 |
| 4K テレビ | 約8M画素 |

つまり**MacBook Pro 16インチの2倍以上の画素数**が目の前に広がる。ピクセルが見えないため、空中に浮かぶテキストはまるで現実の紙に印刷されているかのよう。

### EyeSight技術：外から目が見える

Vision Pro装着者の目が、外部のディスプレイに透けて映る**EyeSight**機能：

| 状態 | 外側ディスプレイ表示 |
|------|----------------|
| 通常作業 | 目の映像（自然な視線） |
| 動画視聴中 | ぼんやりした目（集中中の合図） |
| 完全没入VRモード | カラフルな波形（話しかけ控えて） |
| カメラ使用中 | 目がシフト（撮影中の合図） |

これにより周囲の人と自然なアイコンタクト可能。「ゴーグルを付けた人」ではなく「デジタルとリアルを同時に見ている人」として振る舞える。

## 操作体系：目・手・声の三位一体

### 視線追跡（Eye Tracking）

| 仕様 | スペック |
|------|--------|
| トラッキングカメラ | 4個（赤外線） |
| サンプリングレート | 90Hz |
| 精度 | 視角0.5度以下 |
| キャリブレーション | 初回30秒、自動最適化 |

**「見るだけでフォーカス」**：iPhoneのアプリアイコンを目で見ると、そのアイコンが微妙にハイライト。選択したいウィンドウに視線を向けると自動的にフォーカスが移動。

### ハンドジェスチャー

| ジェスチャー | アクション |
|-----------|--------|
| 親指+人差し指ピンチ | タップ（クリック相当） |
| ピンチしたまま動かす | ドラッグ |
| 両手ピンチして広げる | ズームイン |
| 両手ピンチして縮める | ズームアウト |
| ピンチして指を弾く | スワイプ・スクロール |
| 手首回転 | 3Dオブジェクト回転 |

**手は膝の上でOK**。手を前に出す必要なし。指を曲げるだけで操作完了。長時間でも腕が疲れない設計。

## 内部スペック

### プロセッサ&メモリ

| 項目 | 仕様 |
|------|------|
| メインチップ | M2 |
| 専用チップ | R1（リアルタイム処理） |
| メモリ（M2） | 16GB |
| ストレージ | 256GB / 512GB / 1TB |

### R1チップの役割

R1は12個のカメラ、5個のセンサー、6個のマイクからのデータを**12msの遅延で処理**。これがパススルー（現実映像）の自然な見え方の秘密。

通常のVRは現実映像表示に数十msの遅延があり、酔いの原因に。R1はこれを劇的に削減。
