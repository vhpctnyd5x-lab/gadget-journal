# Apple Intelligence — あなたの生活に溶け込むAI

「AIを使っている感」がない。これがApple Intelligenceの最大の特徴だ。Appleは派手なデモよりも、**日常の小さな摩擦をなくすこと**に集中した。他社のAIが「革命的な新機能」として売り込む一方、Apple Intelligenceは「いつの間にか助けられている」ことを目指している。

## 対応デバイス

| デバイス | 対応 | 全機能利用 |
|---------|------|---------|
| iPhone 16以降 | ✅ | ✅ |
| iPhone 17シリーズ | ✅ | ✅（最速） |
| iPhone 15 Pro/Pro Max | ✅ | △（一部制限） |
| iPad Pro M1以降 | ✅ | ✅ |

A17 Pro / A18 / M1以降のチップが必要なのは、オンデバイスAI処理のためのNeural Engine性能が必要なため。

## Writing Tools：文章を書く時間を半分に

Writing Toolsは**どのアプリでも使えるAI文章アシスタント**。テキストを選択してコンテキストメニューから呼び出す：

| 機能 | 動作 |
|------|------|
| Proofread（校正） | 文法・スペル・表現の誤りを検出・修正案提示 |
| Rewrite（書き直し） | フォーマル/カジュアル/簡潔など文体を変更 |
| Summarize（要約） | 長文を3〜5行に凝縮 |
| Key Points（要点） | 箇条書きで重要ポイントを抽出 |
| Make List（リスト化） | 文章をリスト形式に変換 |

**対応アプリ**：メール、メモ、Safari、Pages、Numbers、Keynote、サードパーティアプリのテキストフィールドすべて。

### 実際の使用例

ビジネスメールでの活用：
1. 会議メモを数行箇条書きで入力
2. Writing Tools → Rewrite → 「フォーマル」を選択
3. 丁寧なビジネスメールが生成される
4. 必要なら微調整して送信

報告書作成：
1. 長い議事録を貼り付ける
2. Writing Tools → Summarize
3. 3行の要点に凝縮
4. そのままメールに転送

## Image Playground：アイデアを瞬時にビジュアル化

テキストプロンプトからオリジナル画像を生成。**完全オンデバイス処理**（クラウド不要）：

| スタイル | 特徴 | 用途 |
|---------|------|------|
| Animation | アニメーション・イラスト風 | プレゼン・SNS |
| Illustration | 手描き・水彩風 | カード・メッセージ |
| Sketch | 鉛筆・スケッチ風 | ラフアイデア共有 |

**写真ベースの生成**：自分の写真を元にしたミー文字スタイルの画像、友人の写真からアニメキャラ風画像を生成できる。

**Photo Cleanup**（写真修正）：背景の不要な人物・ゴミ箱・電柱などをAIが自然に消去。魔法の消しゴムのような機能。

## 強化されたSiri：文脈を理解するアシスタント

新しいSiriは**会話の文脈を記憶して継続的な対話ができる**：

```
「明日の会議の場所は？」→「Tokyo Station近くのカフェです」
「電車で何分かかる？」→（カレンダーと地図を連携して回答）
「向こうの天気は？」→（会議場所の天気を確認）
```

このような自然な流れの会話ができる。以前のSiriは各質問が独立していたため、文脈を保てなかった。

### アプリをまたいだ操作

一言のコマンドで複数アプリを操作：

| コマンド例 | 動作 |
|----------|------|
| 「明日のフライトをカレンダーに追加して」 | メールのフライト情報を解析してカレンダー登録 |
| 「田中さんに今日の会議メモを送って」 | メモアプリの内容を連絡先と照合してメール送信 |
| 「先週撮った富士山の写真を探して」 | 写真アプリを検索して表示 |
| 「今聴いている曲を共有して」 | Musicアプリの現在曲のリンクをシェアシートで表示 |

### 画面の文脈理解（Siri on Mac）

macOS SequoiaのSiriは**表示されている画面の内容を理解**してアクションを提案：
- ターミナルにエラー表示 → 解決策を提案
- Xcodeでコード作成中 → 改善案を提案
- メールに会議の招待 → カレンダーへの追加を確認

## Priority Notifications（優先通知）

AIが通知の重要度を判断して整理する：

| 状況 | 動作 |
|------|------|
| 時間に関係する通知 | 最上部に優先表示 |
| 同じ相手からの複数通知 | グループ化して1件に |
| 広告・プロモーション系通知 | 下位に配置 |
| 緊急性を検出 | 即座にアラート |

通知センターが「情報の洪水」から「重要なことが上にある整理された一覧」に変わる。

## Personal Context Intelligence（iOS 19〜）

iPhoneが保持するすべての情報をAIが横断的に理解：

- **メール + カレンダー**：「明日の会議に間に合う？」→交通情報込みで回答
- **写真 + テキスト**：「先月の大阪旅行の写真から旅行記を書いて」
- **連絡先 + メール**：「田中さんとの最新の会議内容を要約して」

**完全オンデバイス処理**：これらの個人情報はAppleのサーバーに送信されない。

## プライバシー設計：すべてデバイス上で処理

Apple Intelligenceがプライバシーで差別化している理由：

| 処理 | 場所 | Appleへの送信 |
|-----|------|-------------|
| Writing Tools | デバイス上 | なし |
| Image Playground | デバイス上 | なし |
| Priority Notifications | デバイス上 | なし |
| 写真検索・解析 | デバイス上 | なし |
| 複雑なSiriリクエスト | Private Cloud Compute | 暗号化・Appleも閲覧不可 |
| ChatGPT連携（オプション） | OpenAIサーバー | 明示的許可が必要 |

**Private Cloud Compute（PCC）**：クラウドで処理が必要な場合もAppleが内容を見られない設計になっている。PCCではリクエストが完了後にメモリから消去され、ログが残らない。

## ChatGPT連携：Appleのゲートキーパーを通る

Apple IntelligenceはOpenAIのChatGPTと連携できる（オプション機能）：

- Siriが判断して「ChatGPTに転送しますか？」と確認
- 承認後にChatGPTが回答
- **自動転送はしない**：ユーザーの明示的な許可が毎回必要
- ChatGPTへの転送時は個人識別情報を除去

## まとめ

Apple Intelligenceは「使う」というより「いつの間にか助けられている」AIだ。Writing Toolsで文章作業が速くなり、Priority Notificationsで重要な情報を見逃さず、Siriが文脈を理解して作業を代行する。

個人情報がデバイスの外に出ない設計は、他社のAIサービスにない信頼の基盤だ。これがAppleらしいAIの在り方——派手さはないが、確実に日常を快適にする。
