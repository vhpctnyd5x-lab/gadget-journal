# Mac Studio M5 Ultra — スタジオクオリティの制作環境

Mac Studio M5 Ultraは、Appleが「**マシンの性能に制限されないクリエイティブワークフロー**」を実現したデスクトップマシン。M5 Ultraチップは2つのM5 Maxダイを超高速インターコネクト（UltraFusion）で結合した構成で、その性能は文字通り別次元。

¥598,800〜（M5 Ultra）で、Dell Precision 7960（¥1,200,000）と同等以上の性能。消費電力はWindowsワークステーションの半分以下。**プロフェッショナルクリエイターのデスクの中心**になるマシン。

## チップ構成：M5とM5 Ultraの選択

### 仕様比較

| 仕様 | Mac Studio M5 Max | Mac Studio M5 Ultra |
|------|------------------|--------------------|
| プロセス | 3nm（第2世代） | 3nm（第2世代） |
| チップ | M5 Max | M5 Ultra（M5 Max×2） |
| CPU | 16コア（高性能12+効率4） | 32コア（高性能24+効率8） |
| GPU | 40コア | 80コア |
| Neural Engine | 40 TOPS | 80 TOPS（実質） |
| 最大メモリ | 128GB | 192GB |
| メモリ帯域幅 | 546GB/s | 1,092GB/s（推定） |
| トランジスタ数 | 約460億 | 約920億 |
| 価格 | ¥298,800〜 | ¥598,800〜 |

### UltraFusion技術

| 項目 | 仕様 |
|------|------|
| 接続方式 | Apple独自の高密度相互接続 |
| 接続速度 | 毎秒2.5TB |
| レイテンシ | <1μs |
| 動作 | 2つのM5 Maxが「1つの巨大なシステム」として動作 |

M5 Ultraの帯域幅は、ハイエンドNVIDIA GPU（RTX 4090の1,008GB/s）に匹敵。CPU/GPU/メモリの統合は他社追随不可能。
