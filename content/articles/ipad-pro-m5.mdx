# iPad Pro M5 — タブレットの限界を超えて

iPad Proはもはやタブレットではない。**魔法のガラスを持ったプロフェッショナルツール**だ。M5チップとTandem OLEDの組み合わせは、「タブレットでも本格的な制作ができる」という限界をさらに押し広げた。

## ラインナップ：11インチと13インチ

| 仕様 | iPad Pro 11インチ M5 | iPad Pro 13インチ M5 |
|------|-------------------|-------------------|
| ディスプレイ | 11インチ Tandem OLED | 13インチ Tandem OLED |
| 解像度 | 2420×1668（264ppi） | 2752×2064（264ppi） |
| 重量 | 444g | 582g |
| 最大ストレージ | 2TB | 2TB |
| 価格（Wi-Fi/256GB） | ¥178,800 | ¥238,800 |

13インチは携帯デジタルスタジオとしての完成度が高い。11インチは軽さと携帯性を重視するクリエイターに。

## Tandem OLED：ディスプレイ革命

Tandem OLEDは2枚のOLEDパネルを重ね合わせることで輝度を2倍に高め、寿命を延ばした技術だ：

| 仕様 | iPad Pro M5（Tandem OLED） | iPad Air M5（Liquid Retina） |
|------|--------------------------|----------------------------|
| ディスプレイ技術 | Tandem OLED | IPS LCD |
| SDR輝度（持続） | 1,000ニト | 600ニト |
| HDR輝度（ピーク） | 1,600ニト | なし |
| コントラスト比 | 無限大（完全な黒） | 500:1 |
| 局所調光 | ピクセル単位 | パネル全体 |
| 反射防止 | ナノテクスチャガラス（オプション） | 標準 |

**実制作での差**：
- **写真レタッチ**：黒のシャドウ部のグラデーション表現がOLEDで格段に精確
- **映像プレビュー**：HDRコンテンツを撮影した場合の確認が最大1,600ニトで可能
- **夜間使用**：完全な黒が目に優しく、暗所でのスケッチ・読書が快適

**ナノテクスチャガラス（オプション）**：映り込みを極限まで低減する磨りガラス加工。スタジオ照明下でのカラーグレーディング・デザイン確認に最適。(追加¥30,000)

## M5チップ：16GBのパワー

iPad Pro M5搭載のM5チップは16GBのユニファイドメモリを標準搭載：

| 仕様 | iPad Pro M5 | iPad Air M5 |
|------|------------|------------|
| チップ | M5 | M5 |
| RAM | 16GB | 8GB |
| Neural Engine | 38TOPS | 38TOPS |
| Apple Intelligence | ✅（全機能） | ✅（全機能） |

16GBのRAMによる実際の利点：
- Procreate + Adobe Fresco を両方開いた状態でスイッチしてもリロードなし
- Final Cut for iPadで複数の4Kクリップを同時に並べてリアルタイム再生
- LightroomとCapture Oneを同時に起動できる
- 大型アプリ（Logic Pro・Shapr3D）を複数常駐

## Apple Pencil Pro：ペンの究極形

iPad Pro M5とApple Pencil Proの組み合わせはデジタルアートの頂点：

| 機能 | 動作 |
|------|------|
| 筆圧感知 | 4,096段階の精密検出 |
| 傾き検知 | ブラシの太さと向きに反映 |
| バレルロール | ペンを回転してブラシ角度を変更 |
| スクイーズ | 軽く握ってツール切り替え |
| ホバー | 12mm手前でカーソルが追従 |
| 触覚フィードバック | ペン先からリアルな振動 |
| 遅延 | ProMotion 120Hzで体感ゼロ |

Procreateでバレルロールを使った平筆の角度調整、Adobe Frescoでの水彩ブラシの自然な流れ——これらはApple Pencil Proを使って初めて体験できるデジタルアートの新次元だ。

## LiDARスキャナー：ARの可能性

iPad Pro固有の機能がLiDARスキャナーだ（iPad AirにはLiDARなし）：

| 用途 | 具体的な利用 |
|------|-----------|
| AR家具配置 | 実際の部屋に家具を配置してサイズ確認 |
| 3Dスキャン | 物体・空間をスキャンして3Dモデル化 |
| AR測定 | 壁の長さ・天井高をカメラで計測 |
| 建築AR | 工事前の部屋に完成イメージを重ねる |
| 医療AR | 解剖モデルを実空間に重ねた学習 |
| 暗所AF | 暗い環境でも高速・精確なオートフォーカス |

ナイトモード撮影での暗所AF精度向上はLiDARの副産物として特に評価が高い。

## ポート：Thunderbolt 4の接続性

| ポート | 仕様 |
|--------|------|
| USB-C（Thunderbolt 4） | ×1、40Gbps |
| 外付けディスプレイ | 6Kまで対応 |
| 外付けSSD | 最大3,200MB/s |

Thunderbolt 4により：
- 6K外部ディスプレイをiPad Proの「デュアルモニター」として活用
- Thunderbolt SSDへの高速バックアップ（1TBのProResデータを分単位で転送）
- DACやオーディオインターフェースとの接続（音楽制作向け）

## Magic Keyboard：ラップトップを超えるモバイルPC

iPad Pro 13インチ対応のMagic Keyboard：

- バックライトキーボード（暗所でも快適）
- Thunderbolt / USB 4ポート搭載（Proモデル限定）
- ガラスのForce Touchトラックパッド（MacBook品質）
- カンチレバー構造（3〜90度の角度調整）
- 本体充電パス（MagSafe 1ポート）

**Thunderbolt付きMagic Keyboard**の利点：外付けモニター・SSDをMagic Keyboard経由でiPadに接続できるため、デスクセットアップで配線が1本で済む。

| iPadOS機能 | iPad Pro での体験 |
|-----------|----------------|
| Stage Manager | 複数アプリを重なり合うウィンドウで管理（Macライク） |
| Final Cut Pro for iPad | プロ映像編集がiPadで完結 |
| Logic Pro for iPad | 完全版のDAWがiPadで動作 |
| Apple Intelligence | 全機能・最速動作 |
| Multitasking | Split View + Slide Over で最大3アプリ同時 |

**Stage Manager**は特にiPad Pro 13インチで真価を発揮する。13インチの広いディスプレイに複数のウィンドウを重ねて配置することで、Macに近いマルチタスク環境が実現する。

## iPad Pro M5 vs MacBook Air M5：どちらを選ぶか

| 比較 | iPad Pro 13インチ M5 | MacBook Air 13インチ M5 |
|------|--------------------|--------------------|
| 価格（256GB） | ¥238,800〜 | ¥168,800〜（16GB） |
| ディスプレイ品質 | Tandem OLED（最高） | Liquid Retina（良好） |
| ペン入力 | Apple Pencil Pro対応 | なし |
| タッチ操作 | ◎ | △（Touch Barのみ） |
| macOSアプリ | iPadOS（一部制限） | フルmacOS |
| 外部ディスプレイ | 最大1台（Thunderbolt） | 最大3台 |
| キーボード使用時 | Magic Keyboard別売 | 内蔵キーボード |

**iPad Pro M5を選ぶ理由**：
1. デジタルアート・イラストが制作の中心
2. 外出先で持ち運びながら制作する
3. タッチ操作のUIが自然なワークフロー
4. 最高品質のポータブルディスプレイが必要

**MacBook Air M5を選ぶ理由**：
1. Webアプリ・macOS専用ソフトが必要
2. キーボード入力が中心の仕事
3. 複数の外部ディスプレイを使うデスク作業
4. 予算を抑えたい

## カメラ：写真家・映像クリエイター向けの性能

| カメラ | 仕様 |
|--------|------|
| 背面（広角） | 12MP、f/1.8 |
| 背面（超広角） | 10MP、f/2.4 |
| LiDARスキャナー | あり |
| フロント（超広角） | 12MP、センターフレーム対応 |
| ビデオ | 4K/60fps（ProRes記録対応） |

フロントカメラの配置が**横向き**に変更されたことで、Magic Keyboard装着時（横向き使用）でのビデオ通話が自然な画角になった。ZoomやTeamsでの会議クオリティが向上している。

## まとめ

iPad Pro M5は、タブレットとラップトップの境界線を消した。Tandem OLEDの圧倒的な表示品質、Apple Pencil Proとの完璧な統合、16GBRAMによる並列処理能力、LiDARスキャナー——これらすべてが薄いボディに収まっている。

クリエイター、デザイナー、映像制作者にとって、iPad Pro M5は単なる選択肢ではなく「これ1台で完結できるポータブルスタジオ」だ。MacBook Airと比較しても、制作ワークフローに合わせた選択次第でiPad Proが最適解になるシーンは多い。
