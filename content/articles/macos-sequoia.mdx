# macOS Sequoia — MacとiPhoneの境界が消える

macOS Sequoiaは、Macのソフトウェア史において**重要な転換点**だ。単なるUI改善やパフォーマンス向上を超えて、**iPhoneとMacの境界を消す**というAppleの長期ビジョンが形になったバージョン。Apple Intelligenceの本格統合、iPhone Mirroring、ゲームモード——それぞれが、Macの「できること」を根本から広げる。

「PCの時代の終わり」と言われた時期、Appleは「PCを再発明する」道を選んだ。macOS Sequoiaはその答えの一つ。

## iPhone Mirroring：iPhoneをMacの画面で完全操作

### 革命的機能の概要

macOS Sequoia最大の目玉が**iPhone Mirroring**。Macの画面にiPhoneの画面を表示し、Macのマウス・キーボードで完全操作。

| 項目 | 詳細 |
|------|------|
| 接続方式 | Bluetooth + Wi-Fi（ピアツーピア） |
| 遅延 | <50ms（実用上ゼロ） |
| iPhone側画面 | 真っ暗（プライバシー保護） |
| 同時動作 | MacとiPhone独立動作 |
| 必要バージョン | macOS Sequoia + iOS 18 |

### できること

| 機能 | 詳細 |
|------|------|
| **アプリ完全操作** | LINEのチャット、Instagram、TikTokなどスマホ専用アプリをMac上で |
| **ファイル転送** | Mac上の画像・文書をiPhoneアプリに直接ドラッグ |
| **通知の一元管理** | iPhone通知がMacのデスクに表示 |
| **音声出力** | iPhone音声をMacのスピーカーで再生 |
| **クリップボード共有** | Universal Clipboardで一瞬コピペ |
| **ロック維持** | iPhoneロックしたまま操作可能 |

### 実際の使用感

iPhone Mirroringは単なる画面共有ではない。iPhone側の画面は**ブランク（真っ暗）のまま**で、Macからのみ操作できる仕組み。MacのUIとiPhoneのUIが完全独立して動作するため、同時に別々の作業を行える。

#### ユースケース

| シナリオ | 価値 |
|--------|------|
| Macで作業中にLINE返信 | スマホ取り出し不要、集中力途切れず |
| Instagram投稿準備 | Macの大画面でフィード確認・編集 |
| iPhone専用アプリ操作 | 銀行アプリ、フリマアプリなどを大画面で |
| プレゼン中のiPhone操作 | 講演中にiPhoneアプリをスクリーン共有 |
| 開発者のテスト | アプリ開発中の動作確認 |

### Writing Tools：どこでも文章支援

操作方法：選択 → `Control+クリック` → Writing Tools

| 機能 | 動作 |
|------|------|
| 要約 | 長文を3-5行に凝縮 |
| 書き直し（フォーマル） | ビジネス向け表現に変換 |
| 書き直し（カジュアル） | 親しみやすい表現に |
| 書き直し（簡潔） | 要点だけ残して短縮 |
| 校正 | 文法・スペル・表現の改善 |
| 拡張 | 箇条書きを文章に展開 |

#### 実用例

「会議メモを上司向けレポートに変換」：
1. 走り書きの会議メモを選択
2. Writing Tools → 書き直し → フォーマル
3. 整った報告書スタイルに自動変換
4. 微調整して送信

時間：数時間 → 数分。
